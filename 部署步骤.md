# GitHub 部署步骤（你已登录，只需 2 步）

## 第一步：在 GitHub 创建仓库

1. 访问 https://github.com/new
2. 填写信息：
   - **Repository name**: `jingyazhe-portfolio`（或你喜欢的名字）
   - **Description**: 荆亚喆作品集 - 视频剪辑师
   - **Public** ✓（勾选）
   - **Add a README file**: ❌（不勾选）
3. 点击绿色按钮 **Create repository**

## 第二步：复制仓库链接

创建后，在页面找到以下链接格式：

```
https://github.com/你的用户名/jingyazhe-portfolio.git
```

复制这个链接，然后告诉我。

---

## 或者手动推送（如果你熟悉命令行）

创建仓库后，在仓库页面会看到以下命令，复制执行：

```bash
cd /Users/zengningzhi/portfolio-site
git remote add origin https://github.com/你的用户名/jingyazhe-portfolio.git
git branch -M main
git push -u origin main
```
